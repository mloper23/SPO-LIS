# SPO-LIS: Submersible Polarimetric Off-Axis Lensless Imaging System

This repository contains the code, datasets, and validation results associated with the paper:

**"Classification of microplastics in field-collected stream water using a submersible single-shot lensless polarimetric holographic system."**

The proposed system combines **off-axis digital lensless holography**, **single-shot polarimetric acquisition**, and a **lightweight convolutional neural network (CNN)** to detect and classify microplastic particles directly from holographic video streams.

The repository provides tools for:

- Holographic data acquisition  
- Particle detection and patch extraction  
- CNN training and inference  
- Validation of classification results

# System Overview

The SPO-LIS platform performs microplastic identification through the following pipeline:

1. **Four-channel holographic acquisition**  
   Polarization-resolved holographic frames are captured using a polarimetric sensor.

2. **Particle detection**  
   Diffraction signatures in the holographic frames are automatically detected.

3. **Patch extraction**  
   Cropped patches are extracted around each detected particle across the four polarization channels.

4. **Deep learning classification**  
   A lightweight CNN processes each patch and outputs class probabilities for the polymer types.

5. **Particle labeling**  
   Predicted labels are overlaid on the holographic video frames.

---

# Classification Model

The classifier is a lightweight convolutional neural network designed for fast inference on holographic data.

Input tensor:

```
4 × 200 × 200
```

corresponding to the four polarization-resolved holographic channels.

The model outputs probabilities for the following polymer classes:

- **COC** (cyclic olefin copolymer)  
- **PC** (polycarbonate)  
- **PMMA** (polymethyl methacrylate)  
- **PS** (polystyrene)

# Running the Classifier

Example workflow:

1. Load a holographic video  
2. Detect particles  
3. Extract four-channel patches  
4. Run CNN inference  
5. Output labeled particles  

Example command:

```
python classifier/run_classifier.py
```

# Dataset

The dataset contains holographic patches extracted from controlled microplastic samples.

Particles were generated by mechanically grinding polymer pellets and verified using Raman spectroscopy.

Average particle size:

```
3–20 µm
```

Environmental validation experiments were performed using water samples collected from:

**Quebrada La Guayabala, Medellín, Colombia**

---

# Validation

Model performance was evaluated using a train / validation / test split:

```
Train: 64%
Validation: 16%
Test: 20%
```

The trained model achieves classification accuracies up to:

```
~0.98
```

depending on dataset configuration.

Validation metrics and confusion matrices are available in:

```
validation_results/
```
